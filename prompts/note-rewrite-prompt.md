# note版リライトプロンプト

以下の自社サイト記事をnote用にリライトしてください。

## 入力情報
- 元記事: {{article_content}}

## リライトルール（厳守）
1. 完全コピペ禁止。同じ内容でも文章を書き直す
2. 視点の変換：
   - 自社サイトが「ガイド型」（〜の方法、〜の手順）→ noteは「体験記型」（僕が〜した話）
   - 自社サイトが「比較型」（〜おすすめ5選）→ noteは「意見型」（僕が〜を選んだ理由）
3. note版はよりカジュアルに、語りかけるトーンで書く
4. 実体験パート `[EXPERIENCE: ...]` はそのまま残す（人間が埋める）
5. 記事末尾に以下のリンクブロックを必ず入れる：

```
---
📝 詳しい比較や手順は、ブログ記事にまとめています。
👉 [記事タイトル](自社サイトURL)
```

6. 文字数：1500〜2500文字程度（noteで読みやすい長さ）
7. note独自のh2/h3構造は不要。段落ベースで読みやすくする

## 公開タイミング
自社サイトへの公開から1〜2日後に公開すること（インデックス優先権を自社サイトに持たせるため）
