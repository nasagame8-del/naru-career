# ファクトチェックプロンプト

以下の記事について、事実関係のチェックを行ってください。

## 入力情報
- 記事本文: {{article_content}}

## チェック項目

### 1. 数値・統計データ
- 引用されている数値に出典はあるか
- 数値が最新のものか（2年以上前のデータは要注意）
- 「〜万人」「〜%」等の具体的数値は検証可能か

### 2. サービス情報
- 転職エージェントの名称は正しいか
- サービスの特徴説明に誤りはないか
- サービスが現在も運営されているか

### 3. 法的・制度的情報
- 労働基準法等の法律に関する記述は正確か
- 転職に関する制度（雇用保険、失業給付等）の説明は正しいか

### 4. 構造・SEO
- 見出し階層（h1→h2→h3）は守られているか
- faqフロントマターの回答が本文と矛盾していないか
- メタディスクリプション（excerpt）が記事内容を正しく要約しているか

## 出力形式

```
## ファクトチェック結果

### ✅ 問題なし
- [チェック済み項目]

### ⚠️ 要確認
- [確認が必要な項目]: [理由]

### ❌ 要修正
- [修正が必要な項目]: [修正提案]
```
