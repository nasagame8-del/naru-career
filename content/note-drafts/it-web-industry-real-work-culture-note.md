# 飲食業界からIT/Webベンチャーに転職して驚いたこと3つ

「IT業界って未経験でも大丈夫なの？」

転職を考えたとき、僕もまったく同じ疑問を持っていました。大手飲食企業で現場職をやっていた24歳の僕が、AIO対策（AI Overview対策）を手がけるベンチャーに飛び込んでみて、正直に感じたことを書きます。

結論から言うと、未経験でもIT/Web業界には入れます。ただし、飲食業界とはあまりにも文化が違いすぎて、入社後に「マジか」と思った瞬間が何度もありました。

今回は、そのなかでも特に衝撃が大きかった3つを共有します。


**1つ目：誰も「今日やること」を教えてくれない**

飲食時代は、出勤したらその日のオペレーションが決まっていました。指示通りに動けば、それが仕事として成立する世界です。

IT/Webベンチャーに来たら、その前提がまるごとひっくり返りました。「何をやるかは自分で考えてね」が基本スタンスなんです。

最初は本当に戸惑いました。何から手をつければいいかわからない。聞こうにも、みんな自分のタスクに集中していて、手取り足取り教えてくれる雰囲気ではない。GA4やSearch Console、SEO分析ツールといった業務ツールも、全部独学で覚えました。公式ドキュメントを読んで、YouTube見て、実際に触って——というサイクルをひたすら回す日々です。

飲食業界の「マニュアルがあって、先輩が教えてくれる」文化に慣れていた僕には、この自走カルチャーは相当なカルチャーショックでした。


**2つ目：裁量が大きすぎて、全部自分でやることになる**

「裁量権があります」という求人票の文言を見て、ポジティブに受け取っていました。でも実態は「裁量が大きすぎる」でした。

僕の業務は、テレアポ、商談、クロージング、契約、コンサルティングまでを一人で担当しています。営業の全プロセスを一気通貫でカバーする形です。さらにそこに自社のマーケティング戦略まで加わる。

飲食時代は分業が当たり前だったので、「全部やる」というスタイルには正直面食らいました。成長機会としては最高ですが、キャパオーバーと隣り合わせの毎日です。

ちなみに、生成AI（Claude）も業務で日常的に使っています。コンテンツ企画やリサーチの効率化に欠かせないツールですが、これも誰にも教わらず自分で使い方を覚えました。IT/Web業界では「自分で見つけて、自分で使いこなす」が当たり前なんです。


**3つ目：福利厚生の差に、将来のことを考えると不安になる**

これは入社前にもっと真剣に向き合っておくべきだったと感じています。

前職は大手だったので、住宅手当も家族手当も退職金制度も「あって当然」でした。ベンチャーに来てみたら、そういった制度がほぼ整備されていない。24歳の今はまだ困っていませんが、結婚や住宅購入を考えたとき、ライフプランが組みにくいと感じる場面は正直あります。

「やりがいと成長」を取るか、「安定と制度」を取るか。この二択にどう向き合うかが、ベンチャー転職の判断軸になると僕は思っています。


ここまで読むとネガティブな面が目立つかもしれませんが、僕はこの業界に来て後悔していません。

生成AIの時代にマーケティングの最前線に立てていること。自分の裁量で仕事を動かせること。プログラミングなしでも、ノーコードツールだけで成果を出せる環境があること。これは前職では絶対に得られなかった経験です。

ただ「とりあえずIT業界に行けば安泰」みたいな考えだけは、やめた方がいいです。自由と不安定は常にセットです。そこを理解した上で飛び込むなら、未経験でも挑戦する価値は十分あります。

僕みたいに異業種から来る人には、「自分は何を優先して働きたいのか」を転職前にはっきりさせておくことを強くおすすめします。それが定まっていれば、入社後のギャップも前向きに受け止められるはずです。
