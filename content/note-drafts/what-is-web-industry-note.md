# 僕がWeb業界を選んだ理由と、入ってみて分かったこと

「Web業界って実際どうなの？」「プログラミングできないと無理でしょ？」——転職を考えていた頃、僕もずっとそう思っていました。でも実際に飛び込んでみたら、想像とはかなり違っていました。今日は、プログラミング経験ゼロの僕がなぜWeb業界を選んだのか、そして入ってから何が分かったのかを書いてみます。

僕がWeb業界に興味を持ったきっかけは、生成AIの登場でした。ChatGPTが話題になり始めた頃、「これからの時代、AIとマーケティングの交差点にいる人が強くなる」と直感的に感じたんです。でもAIそのものを開発する技術力は僕にはない。それなら、AIを「使う側」として最前線に立てる場所はどこだろうと考えたとき、たどり着いたのがWeb業界でした。

とはいえ、当時の僕にはWeb業界の知識がほぼありませんでした。「Web業界＝エンジニアの世界」だと思い込んでいたし、プログラミングができない自分には縁のない場所だと感じていました。

ところが調べてみると、Web業界にはエンジニア以外の職種が驚くほどたくさんあったんです。Webマーケター、ディレクター、営業、カスタマーサクセス——プログラミングスキルが必須じゃないポジションが普通にある。しかもWeb業界は慢性的に人手不足で、特にAIO対策（AIの検索結果に最適化する施策）のような新しい領域は経験者自体がほとんどいない。つまり、未経験でも入り込む余地が十分にあったわけです。

結局僕は、AIO対策を手がける企業に営業職として入社しました。正直なところ「まず業界に入ること」を最優先にした判断です。マーケティングがやりたかったけれど、未経験でいきなりマーケターとして採用されるのは現実的じゃなかった。だから営業という入口を選びました。

入社してからは、GA4やGoogle Search Consoleといった分析ツールを独学で覚えました。プログラミングは今も業務上必要になっていません。営業として顧客と話す中でマーケティングの知識が自然と身についていき、今では自社の戦略にも関わるようになっています。振り返ると、「最初の職種にこだわりすぎず、まず業界に入る」という判断は正解でした。

Web業界に入って一番驚いたのは、変化のスピードです。半年前の常識が通用しなくなることもある。でもこれは裏を返せば、「新しい領域に先に飛び込んだ人が有利になる」ということでもあります。経験がないことは不利じゃない。まだ誰も経験者がいない場所では、全員がスタートラインに立っている。そう考えると、未経験であることがむしろチャンスだと思えるようになりました。

働き方の面でも、Web業界は柔軟です。リモートワークやフレックスタイムを導入している企業が多く、裁量権も大きい。ただし、裁量権が大きいということは「自分で考えて動かないと誰も助けてくれない」ということでもあります。受け身でいると正直かなり厳しい。ここは覚悟しておいた方がいいです。

「AIに仕事を奪われるのでは」と心配する人も多いですが、現場にいる感覚で言うと、AIはあくまでツールです。クライアントの課題を理解して、戦略を設計して、実行する。この流れは人間にしかできません。むしろAIを使いこなせる人材の需要は高まっていて、Web業界はその最前線にいます。

もし今「Web業界に興味はあるけど、自分にできるか不安」と感じているなら、僕が伝えたいのは一つだけです。プログラミングができなくても、特別なスキルがなくても、Web業界には入れます。大事なのは「なぜWeb業界に行きたいのか」を自分の言葉で語れること。それさえあれば、道は開けます。
